# TODO component spec

1. Add todo
2. Display todo list
3. Delete todo
4. Edit todo name
5. Set completed
6. Set filter
7. Cleare completed list


### Structure
``` jsx
<Todo>
  <Header>
    <TodoInput />
  </Header>
  <TodoList>
    <TodoItem />
  </TodoList>
  <Footer>
    <TodoFilter />
    <TodoAddons />
  </Footer>
</Todo>
```
